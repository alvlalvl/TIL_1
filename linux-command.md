# Linux Command
>리눅스 명령어들을 정리해봅시다.

-pwd(print working director)
    -현재 작업중인 경로를 출력

 -ls(list)
    -현재 폴더에 있는 파일(하얀색.확장자), 폴더(파랑색/)를 출력
    -`-a`(optional):숨김처리된 파일, 폴더까지 출력
    .: 내자신
    ..:상위폴더
-cd(change directory)
    -폴더이동
    -cd ~ 
    -cd DESKTOP/git
-mkdir(make directory)
    -폴더생성
-touch
    -파일생성
-rm(remove)
    -파일 및 폴더 삭제
    - `-r`(optional) : 폴더를 지우기 위해서 입력해야하는 옵션
    rm -r temp
